MonitAir

React Native app for Android and iOS

MonitAir is a smart app designed to regulate air quality in your home. Using a sensor on a Raspberry Pi, Air Quality, Temperature and Humidity are all monitored and reported. This could then react and help alter conditions if air quality became poor. The app was made in React Native, the back end was written in Python, uses PSQL and interacts with the Raspberry Pi. The front end was written in Typescript and uses a Firebase authentication system.

WeatherNow!

React Native app for Android and iOS

WeatherNow! is a react native front end app that tells you weather conditions in any world city currently. It calls on the https://openweathermap.org/api API for data and allows you to impact and re render the data that is displayed on the screen. The app uses full TDD for utility functions with Mocha and Chai, Formik forms for easy data handling and validation of search queries and RN Picker Select to create a custom dropdown country menu to remove client side error.

Forum - NC News

React JS app

NC News is a Reddit esque forum created on React. Written in Javascript it is a React App, with a back end hosted on Heroku. The app uses a RESTful API and has a dummy login system. The front end is syled with CSS and the app is hosted on Netlify. The app was made with TDD using Mocha and Chai.

Film Review App

React Native app for Android and iOS

This film review app allows you add personal reviews and ratings for films you have seen. The app uses routing between pages as well as drawer navigation. I created some custom UI for buttons and cards for a better user experience. I also used modals and Formik forms for validation of input.

Task app

React Native app for Android and iOS

This is a to-do app built using the new React feature of Context and Hooks. I have also used Reducers on it as I wanted to learn and practice with what I think is going to be a major developer tool over the next few years. The app uses local storage to store tasks and is hosted on Netlify.
